# Technical Specification

## Table of contents

- [1. Introduction](#1-introduction)

The purpose of this project is tocreate a virtual school of B3 in 3D, the future premises of ALGOSUP. The ultimate goal of this project is to be able to visit with a virtual reality headset (Oculus Quest 2) and to be able to test all possible movement problems in order to adapt the layout of the building before moving in. The virtual configuration should be as close as possible to the real configuration.
